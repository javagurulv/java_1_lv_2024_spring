Jums ir uzdots izstrādāt API (lietojumprogrammas lietotāja interfeisu)
un tās ieviešana, lai piekļūtu bankas klientu bāzei.
Izmantojot šo saskarni, banka vēlas nodrošināt
ārējo sistēmu iespēja nosūtīt vaicājumus uz savu klientu datu bāzi.

Galvenā prasība ir drošība.
Tikai lietotāji, kuriem būs īpaša loma
(jums tas ir jāizveido) varēs veikt pieprasījumus, izmantojot jūsu
jauna API.

Lai aprakstītu lomas, izveidojiet sarakstu:

enum Role {

    CAN_SEARCH_CLIENTS  // klientam ir jābūt šai lomai, lai to izdarītu
                        // lai viņš varētu meklēt klientus
                        // bankas datu bāzē

}

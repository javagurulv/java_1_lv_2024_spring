Izveidojiet palīdzības kodu Shape hierarhijai.

Izveidojiet ShapeUtilTest klasi un rakstiet testus šādām metodēm:

    double calculateArea(Shape[] shapes);
    double calculatePerimeter(Shape[] shapes);

PS: ģeometrisko formu masīva izveides un aizpildīšanas piemērs:

ShapeUtil shapeUtil = new ShapeUtil();
Shape[] shapes = new Shape[4];
shape[0] = new Circle(4);
shape[1] = new Square(4);
shape[2] = new Rectangle(4, 4);
shape[3] = new Triangle(4, 4, 4);

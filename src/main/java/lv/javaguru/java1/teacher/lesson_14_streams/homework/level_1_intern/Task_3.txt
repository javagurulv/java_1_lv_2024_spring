Zemniekam ļoti patika metode findGreenApples().
Viņa nākamā prasība:
atrodiet visus sarkanos ābolus savā noliktavā.

Naiva pieeja ir izveidot findGreenApples() metodes kopiju
tikai, lai atrastu sarkanos ābolus, atrodiet RedApples():

class FruitStorage {

    public List<Apple> findGreenApples(List<Apple> inventory) {
        List<Apple> result = new ArrayList<>();
        for(Apple apple : inventory) {
            if("green".equals(apple.getColor())) {
                result.add(apple);
            }
        }
        return result;
    }

    public List<Apple> findRedApples(List<Apple> inventory) {
        List<Apple> result = new ArrayList<>();
        for(Apple apple : inventory) {
            if("red".equals(apple.getColor())) {
                result.add(apple);
            }
        }
        return result;
    }

}

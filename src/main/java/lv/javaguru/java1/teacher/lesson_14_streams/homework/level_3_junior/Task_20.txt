Īstenojot uzdevumu no iepriekšējā uzdevuma, jūs nonācāt pie šādas metodes:

public List<Transaction> findTransactionsIn2011(List<Transaction> transactions) {
    return transactions.stream()
                       .filter(t -> t.getYear() == 2011)
                       .collect(toList());
}

Uzrakstiet automātiskos testus OddNumber klases metodēm:

    boolean isOdd(int number) {...}

Lai to izdarītu, izveidojiet klasi OddNumberTest.

Risinājuma izklāsts:

class OddNumberTest {

    public static void main(String[] args) {
        OddNumberTest test = new OddNumberTest();
        test.shouldBeOdd();
        test.shouldNotBeOdd();
    }

    public void shouldBeOdd() {
        OddNumber oddNumber = new OddNumber();
        boolean isOdd = oddNumber.isEven(11);
        if (isOdd) {
            System.out.println("TEST IS OK!");
        } else {
            System.out.println("TEST IS FAIL!");
        }
    }

    public void shouldNotBeOdd() {
        OddNumber oddNumber = new OddNumber();
        boolean notOdd = oddNumber.isEven(12);
        if (notOdd) {
            System.out.println("TEST IS OK!");
        } else {
            System.out.println("TEST IS FAIL!");
        }
    }
}


Mērķi:
- iepazīties ar automatizēto testu rakstīšanu;
- iemācīties ģenerēt testpiemērus;
- trīs soļi automatizēta testa rakstīšanai:
     testa datu sagatavošana,
     izsaucot ražošanas kodu,
     pārbaudot rezultātu.
